## 三、详细功能需求

### （一）核心操作功能

#### 1. 存物品功能



* **功能目标**：支持用户录入物品信息，生成唯一标识标签，通过选择已有储物格或新增储物格（含打印）完成绑定存储

* **前置条件**：

1. 用户已登录系统

2. 家用打印机已连接并正常工作（需打印标签时，也支持跳过打印步骤）

* **操作步骤**：

1. 用户点击首页「存」功能按钮，进入存物品页面

2. 调用物品录入接口：上传物品图片 **或** 手动输入物品文字描述（二选一）

3. 点击「识别并生成标签」按钮，系统处理后展示生成的标签预览

4. 用户确认标签信息无误后，点击「打印标签」（可选：支持直接跳过打印，仅生成标签）

5. 打印完成后，用户选择储物格绑定方式：

* 方式 A（选择已有储物格）：点击「扫描储物格标签」，调用设备扫码功能，扫描已有的储物格标签

* 方式 B（新增储物格）：点击「添加新储物格」，输入储物格名称→点击「生成并打印储物格标签」→打印完成后，扫描该新生成的储物格标签

1. 扫描成功后，系统绑定物品标签与储物格标签，提示「存储成功」

2. 用户放入物品，流程结束

* **输入项**：


  * 物品信息：物品图片（支持 JPG/PNG 格式，≤5MB）或物品文字描述（≤100 字）

  * 储物格信息（新增时）：储物格名称（如「A 区 - 1 层 - 001 格」，≤50 字）

  * 扫码输入：储物格标签的二维码 / 条形码（已有或新增后）

* **输出项**：


  * 物品标签预览（含物品名称、物品编号、二维码 / 条形码）

  * 储物格标签预览（新增时，含储物格名称、储物格编号、二维码 / 条形码）

  * 打印指令（发送至绑定打印机，含物品标签、新增的储物格标签）

  * 存储成功提示（含物品名称、储物格编号 / 名称）

  * 标签绑定记录 + 新增储物格记录（存入系统数据库）

* **业务规则**：

1. 物品编号自动生成，全局唯一（规则：前缀 + 时间戳 + 随机数，如 ITEM-20240520-10086）

2. 储物格编号自动生成，全局唯一（规则：GRID - 时间戳 - 随机数，如 GRID-20240520-00123）

3. 物品名称优先通过图片识别生成，识别失败时使用用户输入的文字描述

4. 二维码 / 条形码包含对应编号、名称、生成时间等核心信息，支持反向扫描查询

5. 未完成储物格绑定的物品标签，系统标记为「未存储」，归到“正在乱放”分类标签下

6. 新增的储物格标签生成后自动存入系统，支持后续查询、补打和重复绑定

* **异常处理**：

1. 图片识别失败：提示「无法识别物品，请手动输入文字描述」

2. 打印机未连接：提示「未检测到可用打印机，请检查连接后重试」

3. 储物格标签扫描无效：提示「该储物格标签不存在或已失效，请重新扫描」

4. 绑定失败：提示「网络异常，绑定失败，请重试」

5. 新增储物格名称重复：提示「该名称的储物格已存在，请更换名称」并提供存入选项，让用户选择是否存入其中

#### 2. 找物品功能



* **功能目标**：支持用户通过多种方式快速查询物品存放位置，关联分类展示

* **前置条件**：

1. 用户已登录系统

2. 系统中存在已存储的物品记录

* **操作步骤**：

  2.1 分类浏览 + 搜索流程：

1. 用户点击首页「找」功能按钮，跳转至分类页面

2. 页面展示所有分类信息卡片（含分类名称、关联物品数量）

3. 用户可点击分类卡片筛选该分类下的物品，或直接在搜索栏输入物品名称

4. 输入完成后点击「搜索」，系统展示匹配的物品列表（含物品名称、物品编号、存储时间）

5. 用户点击目标物品，显示详细信息（含存放位置、储物格编号 / 名称、标签预览）

   2.2 扫码查询流程：

6. 用户点击首页「找」功能按钮，跳转至分类页面

7. 页面点击「扫码查询」按钮，调用设备扫码功能

8. 扫描物品的二维码 / 条形码，系统查询该物品最近一次存储记录

9. 展示物品存放位置、储物格编号 / 名称、存储时间等信息

* **输入项**：


  * 搜索输入：物品名称关键词（≤50 字，支持模糊匹配）

  * 扫码输入：物品标签的二维码 / 条形码

* **输出项**：


  * 分类信息卡片（含分类名称、关联物品数量）

  * 物品搜索结果列表（含物品名称、物品编号、存储时间）

  * 物品详情（含存放位置、储物格编号 / 名称、标签预览、存储时间）

  * 扫码查询结果（直接展示物品详情）

* **业务规则**：

1. 分类信息与物品自动关联，存储物品时系统自动匹配分类（或用户手动选择）

2. 搜索结果按「存储时间倒序」排列，优先展示最近存储的物品

3. 扫码查询仅返回该物品「最后一次」存储的位置信息

4. 无匹配结果时，显示「未找到该物品记录，请检查输入或扫码是否正确」

* **异常处理**：

1. 搜索无结果：提示「未找到包含该关键词的物品，请更换关键词重试」

2. 扫码无效：提示「该物品标签不存在或已失效，请确认标签正确」

#### 3. 打印标签功能



* **功能目标**：支持独立打印物品标签和储物格标签，满足新增、补打需求

* **前置条件**：用户已登录系统，打印机已连接并正常工作

* **3.1 打印物品标签（独立模式）**：


  * 操作步骤：

1. 用户点击首页「打印标签」→ 选择「物品标签」

2. 上传物品图片或输入物品文字描述

3. 点击「生成并打印」，系统识别物品并生成标签

4. 发送打印指令，提示「打印成功」

* 输入项：物品图片（JPG/PNG，≤5MB）或文字描述（≤100 字）

* 输出项：物品标签打印件（含名称、编号、二维码 / 条形码）

* 业务规则：与「存物品」中的标签生成规则一致，支持重复打印（同一物品编号可多次打印）

- **3.2 打印储物格标签**：


  * 操作步骤：

1. 用户点击首页「打印标签」→ 选择「储物格标签」

2. 输入储物格名称（如「房间 - 书桌柜 - 001 格」，≤50 字）

3. 点击「生成并打印」，系统自动生成标签

4. 发送打印指令，提示「打印成功」

* 输入项：储物格名称（≤50 字，支持批量输入，用逗号分隔，一次最多 10 个）

* 输出项：储物格标签打印件（含储物格名称、储物格编号、二维码 / 条形码）

* 业务规则：

1. 储物格编号生成规则与「存物品」中新增储物格一致，全局唯一

2. 支持批量打印，输入多个储物格名称时，按顺序生成对应标签并连续打印

3. 储物格标签生成后自动存入系统，支持后续查询和补打

* **异常处理**：

1. 批量输入格式错误：提示「批量输入格式有误，请用逗号分隔多个储物格名称」

2. 打印机故障：提示「打印失败，请检查打印机状态后重试」

3. 图片 / 文字输入无效：提示「请上传有效图片或输入合法文字描述」

4. 储物格名称重复：提示「该名称的储物格已存在，是否继续打印（编号不变）？」（支持确认后补打）

### （二）分类管理功能



* **功能目标**：支持物品分类的维护与展示，辅助找物品功能高效筛选

* **操作步骤**：

1. 管理员进入「系统设置」→「分类管理」页面

2. 支持「新增分类」：输入分类名称（≤20 字）、分类描述（≤100 字），点击保存

3. 支持「编辑分类」：修改分类名称、描述，点击更新

4. 支持「删除分类」：确认后删除该分类（需先解绑该分类下的所有物品）

5. 分类页面默认展示所有有效分类的信息卡片

* **业务规则**：

1. 分类名称全局唯一，不可重复

2. 默认分类：系统初始创建「默认分类」，不可删除

3. 物品存储时，若未手动选择分类，自动归入「默认分类」

* **异常处理**：

1. 分类名称重复：提示「该分类已存在，请更换名称」

2. 删除绑定物品的分类：提示「该分类下仍有关联物品，请先转移物品后再删除」

### （三）标签管理功能



* **功能目标**：记录所有生成的标签信息，支持查询、导出、补打

* **操作步骤**：

1. 用户进入「标签管理」页面，选择「物品标签」或「储物格标签」

2. 支持按关键词搜索（物品 / 储物格名称、编号）、按时间筛选（生成时间）

3. 列表展示标签信息（名称、编号、生成时间、状态、操作）

4. 支持「补打标签」：点击操作栏「补打」，直接发送打印指令

5. 支持「导出记录」：导出 Excel 格式的标签列表（含编号、名称、生成时间、关联信息）

* **业务规则**：

1. 标签状态分为「已使用」（绑定储物格 / 物品）、「未使用」（仅生成未绑定）、「已失效」（手动标记）

2. 标签记录永久保留，支持按时间范围筛选导出

3. 储物格标签包含「存物品」流程中新增生成的标签，统一纳入管理

* **异常处理**：

1. 导出失败：提示「网络异常，导出失败，请重试」

2. 补打失败：提示「打印机未连接或标签已失效，无法补打」

### （四）基础支撑功能

#### 1. 登录认证功能



* 功能目标：验证用户身份，控制系统访问权限

* 操作步骤：输入用户名、密码，点击「登录」，验证通过后进入首页

* 业务规则：支持管理员 / 普通用户角色，管理员拥有分类管理、标签删除等权限

* 异常处理：账号密码错误，提示「用户名或密码不正确，请重试」

#### 2. 打印机管理功能



* 功能目标：支持打印机连接检测、打印参数设置

* 操作步骤：进入「系统设置」→「打印机管理」，选择已连接的打印机，设置打印份数（1-10 份）、标签尺寸（默认 A4 / 自定义）

* 异常处理：未检测到打印机，提示「请连接打印机并刷新页面」

## 四、非功能需求概要



1. 响应速度：页面加载时间≤2 秒，标签生成时间≤3 秒，搜索查询时间≤1 秒

2. 兼容性：支持 Chrome、Edge、Firefox 最新版浏览器，支持电脑端、平板设备访问

3. 安全性：用户密码加密存储，标签信息支持权限控制，仅授权用户可查看

4. 稳定性：系统连续运行无故障时间≥72 小时，打印功能无卡纸、漏打等问题

5. 易用性：操作流程简洁，关键步骤有引导提示，支持新手教程弹窗

## 五、补充说明



1. 物品识别功能需集成第三方 OCR 接口，若识别准确率不足，可提供手动修改物品名称的入口

2. 扫码功能支持电脑摄像头扫码、外接扫码枪扫码两种方式

3. 标签打印尺寸默认支持 50mm×30mm、80mm×50mm 两种常用规格，支持自定义尺寸输入

4. 「存物品」流程中新增的储物格标签，与独立打印的储物格标签统一管理，无功能差异
